# beer-api

Assignment for a job application.

# Run

```bash
npm run start
```

# Endpoints

Rest api for beer containing the following endpoint:

- GET /beer/
- GET /beer/:id
- POST /beer
- PUT /beer/:id
- DELETE /beer/:id

The payload are in JSON. It make it easier to use [Httpie](https://httpie.io/).
